Projects

ECG Based Biometric Authentication

This project implements human identification from ECG signals via sparse representation of local segments using optmization methods like ADMM and PROXIMAL Algorithm. Achieves a 96.00% accuracy on a 10 subject dataset.

Generation of Celebrity Faces

Face of celebrities are generated using generative adversarial networks (GAN). Generated faces are in the dimension of 64x64. The project is implemented with pytorch.

Sentimental Analysis of Movie Reviews

This project implements machine learning algorithms like Multinomial Naïve Bayes, SVM, XGBoost, Random Forest, Logistic Regression and and LSTM Neural Network for the sentimental analysis of movie reviews. We are able achieve an efficiency of 66% and 89% for the IMDB and the Rotten Tomatoes dataset respectively.

Feature Selection for Classification Using Genetic Algorithm

In this project, we minimise the number of features using Genetic algorithm for classificating the dataset.
